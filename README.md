This project currently targets:

- **Flutter 3.22.x**
- **Dart 3.4.x**

Minimum SDK constraint from `pubspec.yaml`:

- `sdk: ">=3.3.0 <4.0.0"`

If your local version is older/newer in an incompatible way, screens or plugins can fail to run correctly.
